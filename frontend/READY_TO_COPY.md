## 🚀 How to Copy & Deploy

### Step 1: Copy Files
1. Open VS Code file explorer (left sidebar)
2. Navigate to `/app/frontend/`
3. Right-click folder → Download/Copy
4. Paste to your local machine

### Step 2: Install Dependencies
```bash
cd frontend
yarn install
# or
npm install
```

### Step 3: Update Your Information
Edit `src/data/portfolioData.js`:
- Your name
- Your email & social links
- Your skills
- Your experience
- Your projects
- Your bio

### Step 4: Add Your Resume (Optional)
- Place your PDF in `public/resume.pdf`

### Step 5: Test Locally
```bash
yarn start
```
Opens at http://localhost:3000

### Step 6: Build for Production
```bash
yarn build
```
Creates `build/` folder

### Step 7: Deploy
Upload `build/` folder to:
- Vercel
- Netlify  
- GitHub Pages
- Any static hosting

## 🐛 Common Issues & Fixes

### Issue: "Module not found"
**Fix:** Run `yarn install` or `npm install`

### Issue: "Port 3000 already in use"
**Fix:** Kill process or use different port: `PORT=3001 yarn start`

### Issue: "Build fails"
**Fix:** Delete `node_modules` and `yarn.lock`, then `yarn install`
